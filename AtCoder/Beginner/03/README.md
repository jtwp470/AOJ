## B: AtCoderトランプ
条件をきちんと考えればOK.S, Tをchar型の配列(C言語的な考え方)とすれば

* S[i] == T[i] == '@' のときは無条件一致
* S[i] == '@' のときはT[i]にatcoderという文字が含まれていればそのまま処理を続行し,含まれていなければ終了すればいい.これはT[i] == '@'の時も一緒.
* S[i] != T[i] のときは失敗

この方針に従って愚直に実装すれば良い.
